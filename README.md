# Who Owes Me

**WhoOwesMe** helps you keep your finances organized: record debts, payments, and balances easily. Track who owes you, manage transactions, and maintain a clear history of all financial interactions.

## Features

- **Debt Tracking**: Log who owes you (or who you owe), with detailed information.
- **Payments Management**: Record payments made or received, and automatically update balances.
- **Clear Transaction History**: View all debt and payment transactions in a clean, chronological timeline.
- **Balance Overview**: Instantly see how much money you’re owed, or you owe, across all contacts.
- **User-friendly Interface**: Simple, intuitive design built with Kotlin for a smooth experience.

## Getting Started

### Prerequisites

- [Android Studio](https://developer.android.com/studio)
- Android device or emulator (API 24+)

### Installation

1. Clone the repository:
    ```bash
    git clone https://github.com/antonio-macave/who-owes-me.git
    ```
2. Open the project in Android Studio.
3. Build and run the app on your device or emulator.

## Usage

- Add a new debt entry by specifying the person, amount, and description.
- Record payments received or made.
- Track overall balances for each contact.
- Review the full financial history at any time.
